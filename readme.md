# Механик машин (Machine Mechanic)

Расширение для **SillyTavern** — форк Memory Books. Иконка-ключ 🔧 рядом с
волшебной палочкой открывает полный интерфейс приложения **прямо в чате как
сообщение**, целиком на русском. Работает **без** отдельной Memory Books.

## Возможности
- **🔧 интерфейс в чате** — значок у палочки рисует настоящий интерфейс приложения
  как сообщение в ленте; крестик / «Закрыть» убирают.
- **Русский принудительно** — независимо от языка SillyTavern.
- **«?» контекстная справка** — у каждой настройки; клик открывает справку на нужном
  месте с подсветкой.
- **Маркеры сцены ◄►** — в видимой шапке каждого сообщения (не в скрытом меню),
  удобно на телефоне.
- **🌐 Перевод сообщений** — значок у сообщения переводит его через ИИ (перевод под
  сообщением). Свой промпт, профиль подключения SillyTavern, своя температура.
- **🎲 Кубик** — свой рандомайзер; режимы «число» / «успех-провал» (у каждого свой
  промпт); бросок скрыто уходит в промпт (ИИ учитывает исход) и рисуется над
  сообщением; разовый форс следующего броска; шкала критичности.
- **Стрелки ◄► у ножниц** — расширяют выделение текста для «Clip» на соседние сообщения.

## Установка (по ссылке)
В SillyTavern: **Extensions → Install extension** → вставить URL этого репозитория → **F5**.
Обновление — кнопка **Update** у расширения.

Кнопка 🔧 появится внизу, у поля ввода, рядом с волшебной палочкой.

## Разработка / сборка
```
npm install
npm run build   # esbuild: index.js + style.css -> index.build.js + style.build.css
```
`machineMechanic.js` и `help.html` грузятся напрямую — их правки видны сразу (F5),
без пересборки. Пересборка нужна только после правок `index.js` / `style.css`.
